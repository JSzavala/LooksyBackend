Definiciones

| Término | Definición e información | Alias |
|---|---|---|
| Icono | Botón representativo visualmente con una imagen que indica qué función puede tener. |  |
| Artículo | Un artículo textil como ropa, calzado, accesorios, etc. |  |
| Tag | Etiqueta con el que se puede identificar a qué grupo o sección pertenece el artículo presentado. | Hashtag # |
| Post | Publicación dentro de la aplicación, principalmente imágenes que el usuario puede subir dentro de la plataforma para compartir ideas. |  |
| Live | Transmisión en directo (Tiempo real) de artículos a la venta o comprados |  |
| Lista de deseos | Apartado donde se almacenan las posibles compras futuras, quedando como un artículo que deseas comprar. |  |
| Feed | Sección principal donde aparecen las publicaciones en las que el usuario podría estar interesado o aquellas personas a las que tiene en follow. |  |
| Follow | Apartado que permite “Seguir” personas dentro de la plataforma con el fin de que sus publicaciones continúen apareciendo en el feed del usuario quien puede sentirse identificado con el estilo que vio. |  |
| Trending | Artículos más vendidos/valorados positivamente dentro de la plataforma durante cierto periodo de tiempo. | Tendencia |
| Carrito de compras | Apartado en donde se encuentran los productos que se desean comprar, y un paso más cercano a realizar la compra de los artículos. |  |
| Proceso de venta física | Implica realizar la petición de recoger la orden en la tienda física, pero realizar el pedido desde la plataforma. |  |
| Solicitud de venta | Petición de la empresa que desea utilizar la plataforma como intermediaria para la venta de productos. |  |
| Actualización de stock | Apartado en donde es posible que las tiendas realicen modificaciones al stock de los artículos, y los precios de los mismos. |  |
| Historial de compras | Apartado en donde el cliente podrá encontrar el historial de los artículos que ha comprado y los detalles de cada uno. |  |
